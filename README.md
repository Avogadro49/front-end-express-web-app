## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/Avogadro49/front-end-express-web-app
   ```
2. Navigate to the project directory:
   ```bash
   cd front-end-express-web-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run Project
   ```bash
   npm run dev
   ```
